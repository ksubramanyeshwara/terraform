# Terraform

This repository documents my learning journey of Terraform. It will have notes, examples, and small projects to help me understand Terraform better.

## 🎯 Learning Goals

- Understand core Terraform concepts — providers, resources, variables, outputs
- Learn how Terraform manages infrastructure state
- Build reusable infrastructure using modules
- Manage multiple environments using workspaces
- Integrate Terraform into CI/CD pipelines

## 🛠️ Prerequisites

- Terraform installed on your local machine
- Basic knowledge of cloud platforms (AWS, Azure, GCP)
- Familiarity with YAML/JSON for configuration files
- Basic understanding of cloud infrastructure concepts
